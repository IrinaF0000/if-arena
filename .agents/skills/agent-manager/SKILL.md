# Skill: Agent Manager

Use when maintaining AGENTS.md files, skills, task packets, or agent workflow.

## Responsibilities

- Keep root instructions short.
- Move detailed procedures into skills or focused rule docs.
- Add rules only for repeated or high-risk mistakes.
- Avoid duplicated instructions.
- Keep task packets scoped and executable.
- Review token economy and context loading.
- Record harness changes in `docs/agent-manager/`.
