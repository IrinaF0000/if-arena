# Task Packet Rules

Each implementation task must specify:

- goal;
- non-goals;
- allowed files;
- forbidden files;
- security impact;
- performance impact;
- required skills;
- required checks;
- definition of done;
- review gates.

Do not start coding from a vague prompt. Ask the Coordinator Agent to create or refine a task packet first.

Task packets live in `docs/agent-tasks/`. Run notes live in `docs/agent-runs/`.
