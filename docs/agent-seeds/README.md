# Legacy Agent Seed Templates

This directory keeps older reusable templates for reference.

For active work, prefer the scoped harness:

- root `AGENTS.md`;
- nearest nested `AGENTS.md`;
- focused rules in `docs/agent-rules/`;
- repo-local skills in `.agents/skills/`;
- task packets in `docs/agent-tasks/`.

Use these legacy templates only when creating a new task packet or migrating old prompts.
