# Agent Seeds

Reusable templates for optional AI-agent assisted work and seeds for future projects.

These templates are intentionally kept under `docs/agent-seeds/` so the repository root stays clean. If you use active control files locally, either keep them uncommitted or move the final, stable process notes into `docs/agent-workflow.md`.

Suggested local copies:

```text
docs/agent-seeds/AGENT_CONTEXT_TEMPLATE.md  -> local AGENT_CONTEXT.md
docs/agent-seeds/AGENT_RULES_TEMPLATE.md    -> local AGENT_RULES.md
docs/agent-seeds/AGENT_WORKFLOW_TEMPLATE.md -> local AGENT_WORKFLOW.md
docs/agent-seeds/AGENT_PROGRESS_TEMPLATE.md -> local AGENT_PROGRESS.md
```

Convention:

- `docs/agent-workflow.md`: public, stable workflow description.
- `docs/agent-seeds/`: reusable templates.
- local `AGENT_*.md`: optional scratch/control files for active agent work.
