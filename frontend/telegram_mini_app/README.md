# IF Arena Telegram Mini App

Strict TypeScript frontend for the Telegram Mini App client.

Owns:

- Telegram bridge;
- WebSocket client;
- runtime protocol message guards;
- Canvas rendering;
- touch/desktop controls;
- local player-oriented coordinate transforms;
- HUD and connection state.

Must not contain:

- Telegram bot token;
- backend session secret;
- authoritative game rules;
- direct trust in `initDataUnsafe`.

The Mini App sends raw Telegram `initData` to the backend for validation after the WebSocket `open` event and then sends only versioned player intentions such as move, attack, dash, and stop. Objective pickup and capture are automatic server rules. The client responds to server `ping` envelopes with `pong` so passive matched clients remain connected while receiving live snapshots.

Inbound WebSocket data is treated as `unknown` until `src/protocol/ProtocolTypes.ts` validates the protocol envelope.

Canvas rendering loads the production player sprite from `/players/swordsman.svg`, which Vite serves from the repository `assets/` directory. The SVG is used only as rendering data; gameplay objects still come from server snapshots.

## Local commands

```bash
npm run typecheck
npm run lint
npm run build
```

Browser E2E checks require the C++ server to be built first:

```powershell
cmake --build ..\..\build --parallel
npm.cmd run test:e2e
npm.cmd run test:e2e:headed
```

If Playwright browsers are not installed on a fresh machine, run `npx.cmd playwright install chromium`.

The frontend reads `VITE_WS_URL` when provided and otherwise connects to `ws://127.0.0.1:8081/ws` for local development.

Controls:

- `WASD` or arrow keys send movement intentions.
- Direction buttons are screen-oriented for both teams in the player-oriented view: Up moves toward the objective from own base, Down moves back toward own base, and Left/Right stay visually consistent.
- Space sends attack in the current movement direction or the last non-zero direction.
- Shift sends dash in the current movement direction or the last non-zero direction.
- On-screen buttons provide touch fallback.

## Dependency policy

Install dependencies with `npm install` from this directory. `node_modules/` and `dist/` are generated local artifacts and must not be committed. A lockfile may be committed when generated by a dependency change task; task 0016 does not change dependency versions.
