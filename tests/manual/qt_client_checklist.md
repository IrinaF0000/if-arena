# Qt Client Manual Checklist

- [ ] Connect to local server.
- [ ] Create match.
- [ ] Join with second client.
- [ ] Start match.
- [ ] Move with WASD.
- [ ] Aim/attack with mouse or Space.
- [ ] Interact with objective using E.
- [ ] Own/enemy/hazard/objective are visually distinct.
- [ ] Disconnect server and confirm UI remains responsive.
