# Qt Client Manual Checklist

- [ ] Connection state starts as disconnected.
- [ ] Connect to local server.
- [ ] Connection state shows connecting, connected, error, and disconnected states.
- [ ] Create match.
- [ ] Join with second client.
- [ ] Start match.
- [ ] Move with WASD.
- [ ] Aim/attack with mouse or Space.
- [ ] Interact with objective using E.
- [ ] Own/enemy/hazard/objective are visually distinct.
- [ ] Disconnect server and confirm UI remains responsive.
- [ ] Reconnect after disconnect without restarting the client.
- [ ] Invalid/rejected backend command shows user-facing feedback without changing authoritative UI state.
